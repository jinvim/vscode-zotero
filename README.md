# VS Code Zotero

## Features

## Requirements

- Zotero
- Zotero Better Bibtex

## Extension Settings

* `zotero.zoteroDbPath`: Path to Zotero database file (default: `~/Zotero/zotero.sqlite`).
* `zotero.betterBibtexDbPath`: Path to Better BibTex database file (default: `~/Zotero/better-bibtex.sqlite`).
